# Fake_loader by flowside
Fake console **cheat loader**

How to use:
1. Change variables [ _game_name_; _cheat_name_; _target_game_proccess_names_ ] to yours
2. **pip install requirements.txt**
3. Add your code
4. Compile script to **.exe**
